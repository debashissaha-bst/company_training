# Company Training - MySQL Practice Database

A comprehensive MySQL practice project designed for learning SQL from **beginner to advanced** level. Perfect for students, bootcamp participants, and professionals preparing for technical interviews or improving their database skills.

## 📋 About the Project

This repository contains a realistic **company database schema** with multiple related tables covering:

- HR & Employee Management
- Project Tracking
- Sales & Order Management
- Inventory & Products

The database includes rich sample data and a wide range of SQL queries organized by difficulty level.

## ✨ Features

- ✅ Relational database schema (MySQL)
- ✅ Realistic business scenarios
- ✅ Sample data for 8+ tables
- ✅ Queries categorized by difficulty:
  - Beginner (SELECT, WHERE, ORDER BY, etc.)
  - Intermediate (JOIN, Aggregations, GROUP BY)
  - Advanced (Window functions, CTEs, Complex Subqueries, etc.)
- ✅ Run with one script

## 🗂️ Database Schema

### Main Tables:
- `departments`
- `employees`
- `projects` & `project_assignments`
- `attendance`
- `salary_history`
- `categories` & `products`
- `customers`, `orders`, `order_items`, `payments`
